TẠO USB BOOT CÀI ĐẶT HỆ ĐIỀU HÀNH WINDOWS 7, 10

Bước 1: Tải phần mềm Rufus mới nhất bằng cách nhấn vào đây . Sau đó di chuyển xuống mục Download >Sau đó chọn vào link tải Rufus đầu tiên.

Bước 2: Khởi động Rufus vừa tải về, thấy giao diện chính của phần mềm như sau. Lúc này hãy nhớ kết nối USB với máy tính vừa cài đặt Rufus nhé!

Bước 3: Trên giao diện chính của phần mềm tạo USB Boot WIN Rufus, thao tác theo hình hướng dẫn sau:

  • (1): Chọn vào tên USB của .
  • (2): Nhấn vào SELECT để chọn file ISO (file cài đặt) Windows 7 hoặc Windows 10 đã tải về trước đó.
  • (3): Partition scheme: Lúc này cần nhấn tổ hợp phím Alt + E để kích hoạt tính năng ẩn tạo USB 2 phân vùng.
  • (4): Target system: Hệ thống tự động chọn vào BIOS or UEFI.
  • (5): Volume Lable: Đặt tên USB Boot của , gợi ý hãy đặt tên sao cho có ý nghĩa nhất, ví dụ: USB CAI WIN 10.
  • (6): File System: Chọn NTFS (khuyến nghị).

Bước 4: Nhấn START để bắt đầu quá trình tạo USB Boot để cài WIN 7, 10 cho máy tính, laptop.

Lúc này Rufus hỏi có muốn format (xóa dữ liệu) USB hay không?
Hãy nhấn OK để bắt đầu quá trình xóa và tạo USB Boot nhé!

Bước 5: Nếu quá trình tạo USB Boot hoàn tất, sẽ nhận được thông báo READ màu xanh.

Lưu ý: Trong quá trình tạo USB Boot nếu gặp lỗi thì rất có thể file ISO (bộ cài đặt Windows) đã bị lỗi,
hoặc USB đang để chế độ chỉ đọc (tìm thanh gạt trên USB để chuyển sang chế độ đọc/ghi).


Bước 6: Sau khi tạo USB Boot Win 7 hoặc Win 10 thành công, có thể sử dụng chúng để tự cài đặt hệ điều hành!

BOOT USB TRÊN MÁY ẢO VIRTUALBOX

Bước 1: Cần phải nhận dạng được USB đang được sử dụng là ổ đĩa số bao nhiêu. Vào Run, nhập “diskmgmt.msc” để mở Disk Management.
 Hoặc nhấp chuột phải vào My Computer (This PC đối với Windows 8.1/10), chọn Manage, sẽ nhìn thấy Disk Management.

Bước 2:cần phải tạo ra một file đĩa ảo (VMDK) và trỏ nó tới ổ đĩa USB. Sử dụng file đĩa ảo này như một ổ cứng trong VirtualBox.

Mở Command Prompt hoặc tìm kiếm cmd ở nút Start, sau đó nhấp chuột phải và chọn Run as administrator.
Di chuyển đến thư mục cài đặt của VirtualBox, mặc định là C:\Program Files\Oracle\VirtualBox.
Mục đích của việc di chuyển đến thư mục này là để sử dụng lệnh VBoxManage.
Có thể không cần di chuyển mà vẫn sử dụng được lệnh đó bằng cách nhập đường dẫn đầy đủ cho VBoxManage.
Sử dụng lệnh sau để di chuyển đến thư mục cài đặt VirtualBox, thư mục này chứa file VBoxManage: cd%programfiles%\Oracle\VirtualBoxb

Tiếp theo, Cần sử dụng lệnh sau để tạo ra file đĩa ảo: VBoxManage internalcommands createrawvmdk -filename C:\usb.vmdk -rawdisk \\.\PhysicalDrive *
Phải thay đổi số của ổ đĩa mà đã kiểm tra trước đó. * USB của mình là Disk 1 thì mình để PhysicalDrive, nếu Disk 2 thì để PhysicalDrive 2 nhé.
Có thể thay đổi tên và đường dẫn của file đĩa ảo theo ý mình. Ở đây munhf tạo file C:\usb.vmdk.
Nếu chọn tên file có dấu cách thì nhớ đặt nó trogn dấu " " nhé, ví dụ "C:\usb 2.vmdk" .

Nếu thấy thông báo như sau là đã thành công: RAW host disk access VMDK file C:\usb.vmdk created successfully.
